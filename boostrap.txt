Grid vs Flexbox by Maximilian
--------------------------------
https://www.youtube.com/watch?v=RSIclWvNTdQ

Grid is two dimensional. Defines rows and columns, the total layout.
Flexbox is one dimensional and can be used by Grid for a cell or a row.
Grid uses Flex as an extra tool(functionalities) for components' more flexible alignment within a row/cell

Flexbox by Maximilian's colleague
----------------------------------
https://getbootstrap.com/docs/4.0/utilities/flex/

1st part https://www.youtube.com/watch?v=siKKg8Y_tQY
2nd part https://www.youtube.com/watch?v=RdlEEfx912M

Flex container is the parent and flex items are the children
Parent side properties:
  display, flex
  flex-flow, justify-content, align-content, align-items
Using display css property we transform an element to a parent container
Children side properties:
  order, flex, align-self
